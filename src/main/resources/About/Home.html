Supporting

The NIST LRI Test Suite supports the testing of HL7 v2.5.1 messages in support of the LRI Community.

  • HL7 Version 2.5.1 Implementation Guide: S&I Framework Lab Results Interface, Release 1, DSTU Release 2 - US Realm
Have a Question?

A Google Group (HL7v2 Lab Testing) has been established for discussion/questions about the Implementation Guide, testing tool, and testing issues. No membership is required. A Google account is required for posting. When asking questions, please specify the edition of the tool and version of the standard to which the questions apply.

  • Site: https://groups.google.com/d/forum/hl7v2-lab-testing
  • Email: hl7v2-lab-testing@googlegroup.com
